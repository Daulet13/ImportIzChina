## 8. ОБНОВЛЁННЫЕ ВЫВОДЫ ДЛЯ БИЗНЕС-ПРЕДЛОЖЕНИЯ

### Что ИЗМЕНИЛОСЬ после 1С:

| Было (итерация 4, только Логос) | Стало (итерация 5, Логос + 1С) |
|---|---|
| Цены = таможенная стоимость ($0.30-0.68/кг) | Реальные цены = **$1.05-2.80/кг** (в 3-9x выше) |
| Экономия от Китая: 18-73% vs таможенной | Экономия от Китая: **60-90%** vs реальных цен! |
| ANKARA GYD — один поставщик | ANKARA GYD — **три разных контрагента** с разными ценами |
| Неясно, сколько платит Питатель | **5.33 млрд руб** за 5 мес = **$164M/год** |
| 208K тонн из Логоса | **65K тонн** из 1С (более точная цифра за тот же период) |

### Почему 65K тонн (1С) < 208K тонн (Логос)?
- 1С показывает **ОтчётКомитенту** (= финальные проведённые документы)
- Логос показывает **все ДТ** (включая непроведённые, дублированные, другие организации)
- 1С-данные БОЛЕЕ ТОЧНЫЕ для коммерческого анализа

### Пересчёт потенциала экономии:

Если Питатель платит в среднем $1.05-1.50/кг за иранские товары (из ОтчётКомитенту), а Китай предлагает аналогичные по $0.30-0.60/кг (из Логос customs):

- Экономия: **50-80%** (не 18-73% как ранее!)
- На объёме 65K тонн/5мес = **156K тонн/год**
- При средней экономии $0.60/кг × 156,000,000 кг = **$93.6M/год** (грубая оценка)

Даже при переходе **20% объёма на Китай:**
- 31,200 тонн × $0.60/кг = **$18.7M/год экономии**
