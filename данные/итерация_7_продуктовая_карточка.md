## 1. КРИТИЧЕСКИЙ ПОВОРОТ: Китай заменяет НЕ всё

### Три причины ограничений:
1. **Логистика:** 15-25 дней из Китая vs 5-7 из Ирана → нежные овощи не доедут
2. **Фитосанитария:** Россельхознадзор блокирует китайские томаты (18 случаев вируса в 2025)
3. **Вкус/репутация:** азербайджанская хурма = бренд, китайский чеснок = мягче/хуже

### Что НЕ заменить:
| Товар | Объём Питателя | Причина |
|---|---|---|
| Перец сладкий | 6,594 т / 5 мес | Нежный, пестициды, стенка тоньше |
| Баклажаны | 3,304 т | Рекордный % пестицидов |
| Салат Айсберг | 2,836 т | Физически не доедет (срок хранения 14-21 день) |
| Огурцы | 2,846 т | Самый скоропортящийся овощ |

## 5. Вкусовые характеристики — сводка

| Товар | Замена Китаем | Вердикт |
|---|---|---|
| Цитрусовые (мандарины, апельсины, лимоны) | **Хорошая** | Роскачество подтвердило безопасность. Положительные отзывы. |
| Капуста (все виды) | **Хорошая** | Китай — мировой лидер. Без заметных различий. Пекинская — традиционно китайская. |
| Чеснок | **Допустимая** (масс-маркет) | 60-70% российского импорта и так из Китая. Мягче, менее острый. Для школ — ОК. |
| Яблоки | **Осторожно** | Фуджи хранятся отлично, но вкус считается хуже азербайджанских. |
| Морковь | **Допустимая** | Корнеплод, вкус сопоставим. Но Узбекистан ближе. |
| Манго | **Хорошая** | Китай (Хайнань) = качественное манго. На Фуд Сити спрос есть. |
| Клубника | **Рискованная** | Нежный товар. Маржа огромная, но потери тоже. |
| Перец, баклажаны, огурцы | **Не рекомендуется** | Не доедут. Пестициды. Потеря товарного вида. |
| Томаты | **Категорически нет** | Фитосанитарные блокировки. |
| Салат | **Невозможно** | Физически не выдержит транспортировку. |
| Хурма | **Крайне рискованно** | Бренд «азербайджанская хурма». Замена = репутационный удар. |

## 6. Итоговая стратегия (два трека)

### Трек А — Питатель (4 товара)
1. Мандарины (экономия ~65%)
2. Капуста цветная/белокоч (экономия ~80%)
3. Апельсины (экономия до 67%)
4. Чеснок (новая позиция для школьного питания)

### Трек Б — свободная продажа в марте
1. Лимоны (мартовский рост x8.3)
2. Клубника (мартовский рост x7.1, 8 Марта)
3. Манго (постоянный рост)
4. Грибы шиитаке/эноки (нишевый, маржа x10-20)

### Стратегия захода к финдиректору
- НЕ упоминать ANKARA GYD и двойные цены
- Приходить с конкретными ценами от брата
- Позиционировать как «оптимизацию закупок»
- Предлагать 4 конкретных товара с расчётом экономии
