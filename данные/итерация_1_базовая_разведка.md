## 6. Валюта инвойсов

Цены в декларациях указаны в двух валютах:
- **USD** — доллары (курс 78-108 руб. в зависимости от периода)
- **CNY** — юани (курс 11-13.4 руб.)

Примерно поровну по количеству строк. Поле `price` в `list_feacc_gtd` — в валюте инвойса. Поле `sum_curs` в `list_gtd` — курс к рублю.

---

## 7. Ключевые поля для дальнейших запросов

| Таблица | Поле | Назначение |
|---------|------|-----------|
| list_feacc_gtd | feacc_no | Код ТН ВЭД (10 знаков) |
| list_feacc_gtd | feacc_text | Название товара |
| list_feacc_gtd | origin_country_code | Страна происхождения ('CN') |
| list_feacc_gtd | price, price_one_kg | Цена инвойсная, цена за кг |
| list_feacc_gtd | net_weight, quantity, unit | Вес, количество, ед. изм. |
| list_feacc_gtd | price_customs | Таможенная стоимость |
| list_gtd | currency, sum_curs | Валюта и курс к рублю |
| list_gtd | date_clear | Дата выпуска (UTC, +3h для МСК) |
| list_gtd | seller_name | Наименование поставщика |
| list_gtd | id_list_client | ID контрактодержателя (#3295 = Сармант) |
