## ЧАСТЬ 2: ЮРИДИЧЕСКАЯ СТРУКТУРА

### Этап 1: Тестовая фура — через импортёра брата (Вариант 3)

Консенсус Claude + Gemini: **НЕ открывать своё юрлицо для первой фуры.**

Схема:
```
Поставщик (Шаньдун) → Крупный импортёр КЗ (партнёр брата)
→ растаможка Хоргос (пошлина ЕАЭС + НДС КЗ 12%)
→ фура до Москвы → продажа на Фуд Сити "с борта" за кеш
```

- Импортёр берёт $1,000-2,000 за фуру или $0.05-0.10/кг
- Не нужно своё юрлицо, валютный контроль, ЭЦП
- Цель: проверить маршрут, зафиксировать реальные потери, подтвердить прибыль

### Этап 2: Масштабирование — гибрид ТОО (КЗ) + ООО (РФ) (Вариант 4)

Когда тестовая фура покажет результат:

```
Поставщик (Китай)
  → ТОО в КЗ (брата) — импорт, растаможка Хоргос
  → Продажа → ООО в РФ (моё) — экспорт из КЗ по НДС 0%
  → ООО в РФ платит ввозной НДС по ставкам РФ
  → Продажа Питателю / оптовикам с обычным российским НДС
```

**Почему именно так:**
- Питатель = госконтракты = нужен РОССИЙСКИЙ поставщик на ОСНО с НДС
- Казахстанское ТОО напрямую Питателю продавать не может
- ООО в РФ может открыть гражданин КЗ

### КРИТИЧЕСКОЕ: НДС-арбитраж НЕ РАБОТАЕТ

Gemini дважды подтвердила: **сэкономить 8% на разнице НДС (12% КЗ vs 20% РФ) невозможно** в белой схеме.

Механизм ЕАЭС:
1. ТОО в КЗ растамаживает, платит НДС 12%
2. При продаже в РФ — экспорт по НДС 0%, ТОО возвращает свои 12%
3. ООО в РФ ОБЯЗАНА уплатить ввозной НДС по ставкам РФ

**Но есть бонус:** чеснок в РФ = НДС **10%** (не 20%!), что делает его ещё выгоднее.

### Конфликт интересов

Gemini предупредила: работая в Сарманте с доступом к коммерческой тайне (наценки ANKARA GYD 3.5-9.5x), нельзя афишировать свою причастность к грузу. Использовать юрлица и контакты брата.
